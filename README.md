# 🧠 Detector de Imágenes IA

**Proyecto Escolar – Tecnología**

Esta página web permite analizar una imagen y estimar si fue creada con Inteligencia Artificial.

---

## 💡 ¿Cómo funciona?

* Subes una imagen.
* La página la envía a un sistema de análisis (Hive AI).
* Se recibe un porcentaje que indica si es IA.
* Se muestra el resultado en pantalla.

---

## 🛠️ Tecnologías

* HTML
* CSS
* JavaScript
* Hive AI API

---

## ⚠️ Importante

* El resultado es una estimación, no es 100% exacto.
* No se guardan imágenes ni datos.

---

## ▶️ Uso

1. Abrir `index.html`
2. Subir imagen
3. Presionar **Analizar**
