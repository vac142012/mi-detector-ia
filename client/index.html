Detector de Imágenes IA

Proyecto Escolar – Tecnología / Computación

Este proyecto consiste en una página web capaz de analizar una imagen y estimar la probabilidad de que haya sido generada mediante Inteligencia Artificial (IA). Para lograrlo, se utiliza un servicio real que analiza imágenes con modelos entrenados.

¿Cómo funciona?

  • El usuario sube una imagen desde su computador (JPG, PNG, etc.)
  • La página envía la imagen a un modelo de análisis
  • El sistema devuelve un porcentaje de probabilidad
  • Se muestra el resultado en pantalla

Tecnologías utilizadas

  • HTML5: estructura de la página
  • CSS: diseño visual
  • JavaScript: lógica del sistema
  • API de análisis de imágenes (IA)

Importante

El resultado es una estimación, no una verdad absoluta. Las IA pueden equivocarse dependiendo de la imagen.
