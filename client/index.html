Detector de Imágenes IA

Asignatura: Tecnología

Curso: 1° Medio A

Alumnas: Victoria Aránguiz - Victoria Prat

Este proyecto consiste en una página web capaz de analizar imágenes y estimar si fueron generadas por Inteligencia Artificial. Utiliza un modelo de análisis que entrega un porcentaje de probabilidad.

¿Cómo funciona?

  • Subes una imagen desde tu computador
  • La imagen se envía al sistema de análisis
  • Se obtiene un porcentaje de IA
  • Se muestra el resultado en pantalla

Importante

El resultado es una estimación, no una verdad absoluta.
