🧠 Detector de Imágenes IA

Proyecto Escolar – Tecnología / Computación

Esta página analiza imágenes y estima si fueron generadas por Inteligencia Artificial. Usa un modelo real para calcular la probabilidad de imagen sintética.

¿Cómo funciona?

  • Subes una imagen desde tu computador
  • Se envía a un modelo de análisis
  • Se obtiene un porcentaje de probabilidad
  • Se muestra el resultado

Importante

El resultado es una estimación, no una verdad absoluta.
