Detector de Imagen Generada por IA

Esta página forma parte de un proyecto escolar. Permite subir una imagen y consultar un servicio real que estima la probabilidad de que haya sido generada con Inteligencia Artificial.


Nota: El análisis se realiza usando un modelo de detección de imágenes generadas por IA expuesto a través de una API externa. El resultado es una estimación, no una verdad absoluta.
